> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Cap Agent Session Replayer typography at 16px and apply the approved compact 14px case-title stack with consistent 3px metadata spacing.

**Architecture:** Keep the change inside the public package stylesheet so the Vite demo and package consumers receive the same scoped behavior. Lock the contract against compiled CSS, then update the current design document and verify the distributable stylesheet, demo build, and packed artifact.

**Tech Stack:** React 19, TypeScript, scoped CSS, Bun workspace scripts, tsup, Vitest, Vite

it("caps typography and compacts the case title stack", () => {

  expect(navRules).toContainEqual(expect.stringContaining("padding:9px 16px 10px"));
